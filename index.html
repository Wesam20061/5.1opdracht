Welkom op onze website

Een ID kan maar één keer per pagina worden gebruikt omdat het een unieke identificator is. Het helpt om specifieke elementen op een webpagina te vinden. Als meerdere elementen dezelfde ID hebben, zou het verwarrend zijn voor de computer, omdat het niet zou weten welk element het moet selecteren. Het is vergelijkbaar met het hebben van identieke namen in een klas - het zou moeilijk zijn om te weten wie je aanspreekt als iedereen dezelfde naam heeft.
